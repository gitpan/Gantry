Gantry Features

  • code generation with BigTop
  • stand-alone server for rapid development
  • runs on anything that runs Perl
  • plugin architecture
  • runs under mod_perl, Fast-CGI, CGI
  • flexible template system
  • user authentication and sessions
  • built-in CRUD operations
  • form generation and validation
  • DBIx::Class support
  • database and SQL utilities

A web application framework for Perl. Object-oriented design allows for a pragmatic, modular approach to URL dispatching.
